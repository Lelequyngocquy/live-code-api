npm init -y
npm install express cors dotenv ioredis bullmq nodemon pg

    express: Framework nhẹ và phổ biến nhất để xây dựng API server.
    cors: Middleware giúp frontend (mặc dù bài này không yêu cầu code frontend, nhưng API thực tế luôn cần CORS) có thể gọi được API của bạn.
    dotenv: Để load các biến môi trường từ file .env vào process.env.
    ioredis: Một Redis client mạnh mẽ cho Node.js, giúp kết nối và giao tiếp với database Redis.
    bullmq: Thư viện xử lý hàng đợi (queue) dựa trên Redis. Nó rất ổn định, hỗ trợ retries, delay, và xử lý bất đồng bộ — hoàn hảo cho yêu cầu "Code execution must be asynchronous" của Edtronaut.


CREATE SESSION: POST
    http://localhost:3000/code-sessions
    body:  
    {
        "language":  "",
        "source_code":  ""
    }

AUTO SAVE SESSION: PATCH
    http://localhost:3000/code-sessions/your-session_id-received-from-above
    body:
    {
        "language": "javascript",
        "source_code": "console.log('Whatsup, my name is Ngoc Quy');"
    }
    {
        "language": "python",
        "source_code": "print('Hi Edtronaut! Im Le Ngoc Quy!.')"
    }
    or
    {
        "language": "php",
        "source_code": "<?php\necho 'hello, i am LE NGOC QUY! ';"
    }
    or
    {
        "language": "ruby",
        "source_code": "puts 'HELLO I AM LNQ!'"
    }




RUN CODE: POST  
    body: 
    http://localhost:3000/code-sessions/your-session_id-received-from-above-again/run
    {
        "language":  "",
        "source_code":  ""
    }

GET RESULT: GET
    http://localhost:3000/executions/your-execution_id-received-from-above
